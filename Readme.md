# series wished

## O projeto

- Vue.js, Vuex e autenticação com API Rest 
- Aplicação SPA 
- usando Vue.js
- controlar lista de desejo de séries de tv 
- consumir uma API Rest
- manter a sessão no frontend
- usando o token JWT.
